This guide covers everything *behind* the glass: the `yoyopod-runtime`
supervisor, the NDJSON wire every process speaks, and the six domain
workers that touch the real world. Like its sibling, the
[UI System Guide](/ui/), it documents the system as it **is** — honest
divergences included.
