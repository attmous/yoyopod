These are the **normative design specs** for the device UI, final as of
2026-07-15. Each file renders the 240×280 panel 1:1 inside a neutral device
bezel, followed by the token, geometry, and behavior tables an implementer
needs. The acceptance rule: renders are judged against the **RGB565 values
in each token table**, not the CSS hexes.

:::note[These are copied snapshots]
The canonical files live at `device/ui/assets/ui/` in the repository,
alongside [`HANDOFF.md`](https://github.com/attmous/yoyocore/blob/main/device/ui/assets/ui/HANDOFF.md)
(the implementation handoff) and
[`README.md`](https://github.com/attmous/yoyocore/blob/main/device/ui/assets/ui/README.md)
(the decision log). After any change there, refresh these copies with
`npm run sync:mockups`.
:::
