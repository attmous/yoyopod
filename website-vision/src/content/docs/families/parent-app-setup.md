## What you'll need

- The device powered on and showing its Setup screen
- The parent app installed on the parent's phone (app availability TBD)
- A family account for the parent (account model TBD)
- Mobile coverage or Wi-Fi during pairing (connectivity requirements TBD)

## Steps

- Install the parent app and create the family account
- Put the device in pairing mode from its Setup screen
- Pair phone and device (mechanism TBD — code on the glass, QR, or similar)
- First configuration pass: name the device, set the contact whitelist
- Load the first music and stories into the library
- Confirm location sharing settings before the first solo outing

## Tips

- Do the pairing together — the kid sees what the parent sees
- Start the whitelist small: parents and one or two trusted family members
- A second parent or caregiver on the same family (support TBD)
- Bookmark [Parental Controls](/families/parental-controls/) — every setting lives there

## Troubleshooting
