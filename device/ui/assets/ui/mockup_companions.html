YoyoPod — Companion Variants

Three swappable animal companions for the Home stage, drawn in the same flat neubrutalist style as the original Blob: thick 2px ink outlines, flat fills, 2,3 px ink drop-shadow, breathing scale animation, blinking eyes. Body fill is a single token per variant — easy to recolor or swap on themes/events. The Blob remains available as a fourth fallback.

Each variant is one root SVG with a single fill hook on its body path (var(--c-*)). Recoloring = one CSS variable change. Eye groups carry the class .eye-iris so the shared 5.5 s blink animation drives all variants uniformly.

3 · Shared style rules (apply to every variant)

  • Outline: 3 px ink #1B1B1F stroke, stroke-linejoin: round, stroke-linecap: round. (In LVGL the outline can be baked into the sprite.)
  • Drop shadow: filter: drop-shadow(2px 3px 0 ink) — blur 0, follows silhouette, identical to the Blob.
  • Breathing: scale 1.00 ↔ 1.04, 4200 ms ease-in-out, infinite. Applied to the wrapper, not the SVG, so it inherits the drop-shadow's silhouette tracking.
  • Blink: eye groups carry .eye-iris; shared scaleY 1 ↔ 0.1 keyframes (5500 ms) fire on each. transform-box: fill-box keeps the scale anchored to each eye's center.
  • Stage position: wrapper is position: absolute inside .home-stage. Each variant chooses its own top/left within the 240 × 206 stage area to keep the visual center between status bar and deck.
  • Layer order: always above stage bg (z-index: 10), always below Deck and StatusBar.

4 · LVGL render mapping

ApproachCostWhen to pick it
Sprite atlas (recommended) ~135 KB / variant @ RGB565 (32 frames × 110×110 + alpha) — load one variant at a time Default. Cheapest CPU. Animation is just blits. Outline + shadow baked in.
SVG → static PNG + procedural breathe/blink ~6 KB body sprite + per-frame transform If RAM is tight; lose the morph but keep breathe/blink via lv_anim on scale + eye alpha.
Vector primitives (1 ellipse body + 2 ellipse eyes + ...) ~12 objects per variant; per-frame style recalc Only for variants where sprite atlas memory is unavailable. Higher CPU.

Variant switch contract: set_companion(Variant) swaps the active sprite atlas. State (breathe phase, blink schedule) resets to t=0 on switch. Variant choice is persisted in user settings (Setup → Companion).
